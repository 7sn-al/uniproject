ABOUT US

Tricky Tourist is a student-built travel platform created by Hassan Alayan, Ahmad Nasser, and Ayman Bakkar with one goal in mind: making exploration feel smooth, modern, and actually enjoyable instead of clunky and outdated.

We designed this website to showcase famous monuments and destinations in a way that feels clean, interactive, and visually engaging. Every page is built with attention to layout, responsiveness, and user experience, so whether you’re browsing casually or actively exploring locations, the site stays intuitive and easy to navigate.

Behind the scenes, Tricky Tourist uses HTML, CSS, and JavaScript to deliver smooth transitions, dynamic content, and a premium look without unnecessary complexity. We focused on clean design choices, readable layouts, and responsive behavior across different screen sizes.

This project reflects our interest in web development and design, and it represents our effort to combine functionality with aesthetics. Tricky Tourist isn’t just a website, it’s a demonstration of how thoughtful design and smart code can work together to create a better user experience.
